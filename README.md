🏠 The Aroma – Indian Restaurant Landing Page
A modern, responsive Landing Page for an Indian Restaurant designed using HTML, CSS (with variables), and JavaScript. This is a frontend-only project with elegant sections like Hero Banner, About, Menu, Gallery, Reservation, and Location.

🌟 Features
Responsive Design (Mobile, Tablet, Desktop)

Clean, minimalistic UI with rich Indian aesthetic colors.

CSS Variables for theme management.

Scroll animations & smooth transitions.

Image Gallery with lightbox effect.

Simple Reservation Form (No backend).

Interactive Navigation Menu.

🎨 Customization
To change the theme colors, update the CSS variables inside :root in style.css:

css
Copy
Edit
:root {
    --primary-color: #d4af37;
    --secondary-color: #8b4513;
    --accent-color: #ff6b35;
    --text-dark: #2c1810;
    --text-light: #f5f5f5;
    --background-light: #faf8f5;
}
Replace images in /images folder to personalize.
